Справка по приложению "SmartCalc_V3"

Общие сведения

Программа разработана на языке Python версии 3.11. Вся кодовая база находится в папке src. Программа представляет собой десктопное приложение с графическим пользовательским интерфейсом, предназначенное для работы на операционных системах Linux и Mac OS.

Установщик

Программа поставляется с установщиком, который автоматически устанавливает приложение в систему с использованием стандартных настроек, включая путь установки и создание ярлыка на рабочем столе. Установщик разработан с помощью pkgbuild для macOS.

Архитектура

Приложение реализовано с использованием паттерна MVP (Model-View-Presenter). В соответствии с принципами этого паттерна:

  • В коде представлений нет бизнес-логики.
  • В модели и презентере отсутствует код интерфейса.

Ядро калькулятора

Калькулятор использует алгоритм формирования и вычисления польской нотации, а также различные вычислительные функции, подключенные в виде динамической библиотеки на C++. Ядро калькулятора представляет собой обертку на языке Python, и все функциональные возможности калькулятора вынесены в модель. Это позволяет использовать модель независимо от остальных слоев.

Основные функции

Программа поддерживает следующие арифметические операции: сложение, вычитание, умножение, деление, возведение в степень, остаток от деления, унарный плюс и унарный минус. Также поддерживаются скобки для формирования выражений.

Поддерживаемые математические функции: косинус, синус, тангенс, арккосинус, арксинус, арктангенс, квадратный корень, натуральный логарифм и десятичный логарифм.

Дополнительные возможности

  • Программа хранит историю операций, позволяет загружать выражения из истории и очищать историю целиком.
  • История сохраняется между запусками приложения.
  • Поддерживается ввод целых чисел и вещественных чисел в экспоненциальной форме записи.
  • Вычисление происходит после полного ввода выражения и нажатия на символ "=".
  • Программа строит графики функций, заданных в инфиксной нотации с переменной x, с отображением области определения и значений.

Точность и ограничения

  • Проверяемая точность дробной части составляет минимум 7 знаков после запятой.
  • Пользователь может вводить до 255 символов.
  • Область определения и значения функций ограничены числами от -1000000 до 1000000.
